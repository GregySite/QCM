Question 1 : A Sales user is trying to manage Campaign Members for an upcoming networking event. The user can view the Campaign, but not add new Campaign Members or update Member statuses. How can an administrator troubleshoot this problem?
  • A. Create a permission set to allow the user to edit Campaign Members.
  • B. Provide the user access to both Leads and Contacts to edit all Members.
  • C. Make sure the Marketing User Checkbox is checked on the user record page.
  • D. Run a Campaign report and update any Member information via Data Loader.
Réponse correcte : C
i
Pour gérer les membres d’une campagne, l’administrateur doit s’assurer que la case “Marketing User” est cochée dans la fiche utilisateur. Cela active les permissions nécessaires.
